This document covers various topics, including: * **Linux Cheat Sheet**: Common Linux commands and their descriptions. * **Data Types**: Storage sizes and value ranges for different data types in C. * **C Logic Operators**: Operator precedence, description, and associativity. * **C Functions**: PRINTF and SCANF functions, along with general operator behavior. * **Bitwise and Arithmetic Operators**: Including overflow/underflow. * **Control Flow**: Selection statements (if/else, switch) and repetition (while, do-while, for loops). * **Recursion**: Rules and examples like Fibonacci Sequence. * **Arrays and Pointers**: How they work, memory allocation, and their relationship. * **Floating Point Numbers and Doubles**: Precision, error analysis, and comparison. * **Math Library**: Usage for functions like sin, exp, and finding roots. * **Structures**: Grouping variables and their usage. * **Dynamic Memory Allocation**: Stack vs. Heap, malloc, free, calloc, and realloc. * **Characters and Strings**: ASCII representation and string manipulation. * **Big Oh Notation**: Analyzing algorithm efficiency (time and space complexity). * **Searching and Sorting Algorithms**: Linear Search, Binary Search, Selection Sort, Insertion Sort, and Merge Sort. * **Linked Lists**: Data structure, deep vs. shallow copies, and recursive copying.
