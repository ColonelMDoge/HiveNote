This document is a C Language and Linux Course Notes, specifically a cheat sheet. It covers various topics including: * **Linux Commands:** Basic commands for file and directory manipulation, compilation, and submission. * **C Data Types:** Ranges and storage sizes for various integer types. * **C Operators:** Precedence, description, and associativity of logical and bitwise operators. * **C Input/Output:** `printf` for output and `scanf` for input. * **Operators:** Arithmetic, relational, and equality operator precedence and associativity. * **Bitwise Operators:** Definitions and examples. * **Arithmetic Overflow/Underflow:** Explanation and implications. * **Selection Statements:** `if`, `if-else`, and `switch` statements. * **Negative Numbers in Binary:** Representation using 1's and 2's complement. * **Bitshifting and Binary Operators:** Operations and examples. * **Repetition in C:** `while`, `do-while`, and `for` loops. * **Break and Continue:** Control flow statements for loops. * **Functions and Modules:** Definition, parameters, assertion, and header files. * **Recursion:** Concept, rules, and examples. * **Arrays and Pointers:** Storage, indexing, initialization, `sizeof`, passing to functions, and pointer decay. * **Floating Point and Doubles:** Precision, limitations, and error calculations. * **The Math Library:** Usage for trigonometric and other functions, and algorithms for finding roots. * **Horner's Method:** Efficient polynomial evaluation. * **Structures in C:** Compound data types, grouping variables, passing structs, and `typedef`. * **Dynamically Allocated Memory:** Stack vs. Heap, `malloc`, `free`, `calloc`, and `realloc`. * **Characters and Strings in C:** ASCII representation, string termination, `const`, `#define`, `gets` vs. `scanf`, and string manipulation functions. * **Big Oh Notation:** Describing algorithm complexity, growth rates, and various notations (Big-O, Little-o, Big-Omega, Little-omega, Big-Theta). * **Recurrence Relations:** Common forms and their Big Oh equivalents. * **Searching and Sorting:** Linear search, binary search, Selection Sort, Insertion Sort, and Merge Sort. * **Linked Lists:** Structure, insertion, copying (shallow vs. deep), and recursive copying.
